Antojito Maps

Antojito Maps es una plataforma web que permite visualizar promociones y descuentos de restaurantes en un mapa interactivo en la ciudad de Cochabamba.
El sistema permite a los restaurantes publicar ofertas y a los usuarios consultar promociones cercanas de manera rápida y sencilla.

Tecnologías

- Angular
- Typescript
- MySQL
- Google Maps API
